# AxiomFlow

**Description:**  
AxiomFlow is an intelligent, scalable finance dashboard that leverages machine learning for predictive analytics and interactive data visualizations. Built with a modern tech stack, it delivers real-time insights, regression modeling, and a sleek UI for data-driven decision-making.

---

## **Frontend (Client)**  
**Dependencies:**  
```bash
npm i --save react-redux @reduxjs/toolkit react-router-dom @mui/material @emotion/react @emotion/styled @mui/icons-material @mui/x-data-grid
```

**Dev Dependencies:**  
```bash
npm i --save-dev @types/react-dom eslint eslint-config-react-app @types/node
```

**Environment Variable:**  
```env
VITE_BASE_URL=http://localhost:5000
```

---

## **Backend (Server)**  
**Dependencies:**  
```bash
npm i --save express cors pino dotenv helmet morgan logger mongoose-currency @typegoose/typegoose http-status-codes mongoose@~6.10.0
```

**Dev Dependencies:**  
```bash
npm i --save-dev @types/cors @types/express pino-pretty ts-node-dev typescript @types/mongoose-currency
```

---

### **Why AxiomFlow?**  
- **Axiom**: Suggests foundational truth or principle—reflecting data-driven insights.  
- **Flow**: Represents seamless data pipelines, real-time updates, and intuitive UX.  
- Combines predictive analytics with financial intelligence, emphasizing clarity and scalability.

---

**Key Features:**  
✅ Machine learning-powered regression models  
✅ Interactive charts (Recharts integration)  
✅ Modular Redux state management  
✅ Secure REST API with Express & TypeScript  
✅ Material-UI design system with DataGrid  
✅ MongoDB via Mongoose & TypeGoose  
✅ Logging, security, and middleware support
